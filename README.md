## 简介

`ZHStyleButton`是 `UIButton`的子类可以设置 `默认` `高亮` `不可点击`三种状态下面的

* 边框宽度`boardWidth`
* 边框颜色`boardColor`
* 圆角大小`cornerRadius`
* 背景颜色`backgroundColor`
* 文本颜色`textColor`
